## Concepts Covered

### 1. Data Entry

- Manual input of structured employee records
- Use of proper data types (text, date, numeric)

### 2. Formatting

- Date formatting for readability
- Currency/number formatting for salary fields
- Clean tabular layout for better visualization

### 3. Cell Referencing

This practical focuses on understanding:

- **Relative Reference**
  - Automatically changes when formula is copied
  - Example: `=C2 + D2`

- **Absolute Reference**
  - Fixed reference using `$` (e.g., `$C$2`)
  - Useful when a constant value is used in multiple calculations

- **Mixed Reference**
  - Combination of relative and absolute (`$C2` or `C$2`)

### 4. Formula Usage

- Calculation of **Total Salary** using:

  ```
  = Salary + Bonus
  ```

- Demonstrates how formulas dynamically update when values change
